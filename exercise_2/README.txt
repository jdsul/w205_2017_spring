Execise 2 README  

To run this application:

1. Change the directory to /w205_2017_spring/exercise_2/extweetwordcount

Note you’ll have to add whatever directory levels are above /w205_2017_spring to the beginning of the path above.

2. type “sparse run”

3. The application will run until killed by the user. To kill the application press CTRL+C (Windows) or option+C (MacOS)

4a. To query the data go to postgres and connect to the tcount database. The data will be stored in the tweetwordcount table in that database.

4b. To run the serving layer scripts change your directory to /w205_2017_spring/exercise_2/extweetwordcount/serving_scripts and execute either of the scripts there.
